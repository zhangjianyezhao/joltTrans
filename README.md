# joltTrans

Browser-based JOLT transformer published with GitHub Pages.

Supported in this first web version:

- `shift`
- `default`
- `remove`
- `cardinality`
- `sort`
- custom `com.tcs.bancs.bancsone.coex.impl.jolt.SystemDate`
